📈 Statistical Analysis

Key Statistical Measures

Mean (Average):
x̄ = (x₁ + x₂ + ... + xₙ) / n

Standard Deviation:
σ = √[Σ(xᵢ - x̄)² / n]

Percent Error:
Error% = |measured - expected| / expected × 100%

📉 Linear Regression & Curve Fitting

When you plot Temperature vs Distance, you often get a straight line (or close to it). Linear regression finds the "best fit" line through your data points and gives you its slope and intercept.

Linear Equation: y = mx + b

Where:
• m = slope (temperature gradient)
• b = y-intercept (extrapolated temperature at x=0)
• R² = correlation coefficient (how well data fits the line)

📊 Creating Professional Graphs

Essential Elements of a Good Graph:

Your Graph Must Include:

  • Title: Descriptive title (e.g., "Temperature vs Distance for Aluminum Bar")
  • Axis Labels: Include units! (e.g., "Distance (cm)", "Temperature (°C)")
  • Legend: If multiple data series, label each one
  • Grid Lines: Helps read values accurately
  • Data Points: Mark actual measurements with symbols
  • Best Fit Line: Show the trend if applicable
  • Equation: Display the line equation and R² value

Using Your App's Graphing Features:

  • Open Graph: Opens separate window for detailed view

⚠️ Error Analysis & Uncertainty

No measurement is perfect! Understanding and quantifying errors is crucial in engineering.

Types of Errors:

Random Errors

  • Unpredictable fluctuations
  • Can be reduced by averaging
  • Example: Temperature sensor noise

Systematic Errors

  • Consistent bias in measurements
  • Require calibration to fix
  • Example: Sensor offset by +2°C
Propagation of Uncertainty:
For Q = k × A × ΔT / d:
δQ/Q = √[(δk/k)² + (δA/A)² + (δΔT/ΔT)² + (δd/d)²]

📝 Writing Lab Reports

Standard Lab Report Structure:

  • 1. Title: Experiment name and date
  • 2. Objective: What you're trying to find out
  • 3. Theory: Relevant equations and concepts
  • 4. Apparatus: Equipment used (your heat transfer apparatus)
  • 5. Procedure: Step-by-step what you did
  • 6. Data: Raw measurements in tables
  • 7. Analysis: Calculations, graphs, processed data
  • 8. Results: Key findings and values
  • 9. Discussion: Interpretation and error sources
  • 10. Conclusion: Summary and learning outcomes
